# Sardauna Tech Lab - Modern Digital Infrastructure

Building the future of African business through innovative digital solutions and automation.

## Tech Stack

- **Frontend**: React 19 + Vite
- **Styling**: Tailwind CSS 4 (Vanilla CSS Integration)
- **Animations**: Motion (Framer Motion v12)
- **Icons**: Lucide React
- **Backend/Database**: Firebase (Firestore, Auth, Storage)
- **Error Handling**: Global Error Boundary Architecture
- **Type Safety**: Centralized TypeScript Interfaces

## Key Features

- **Premium UI/UX**: Smooth animations, glassmorphism, and responsive design.
- **Real-time Data**: Integrated with Firestore using custom hooks for seamless updates.
- **Scalable Infrastructure**: Built with a clean, modular architecture.
- **Robustness**: Global error boundaries and comprehensive type safety.

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up environment variables in `.env`:
   - `VITE_FIREBASE_API_KEY`
   - `VITE_FIREBASE_AUTH_DOMAIN`
   - `VITE_FIREBASE_PROJECT_ID`
   - `VITE_FIREBASE_STORAGE_BUCKET`
   - `VITE_FIREBASE_MESSAGING_SENDER_ID`
   - `VITE_FIREBASE_APP_ID`
4. Run development server: `npm run dev`

## Project Structure

- `src/hooks`: Custom hooks for data fetching and business logic.
- `src/components`: Reusable UI components and layouts.
- `src/pages`: Main page components.
- `src/types`: Application-wide type definitions.
- `src/utils`: Helper functions and utilities.
